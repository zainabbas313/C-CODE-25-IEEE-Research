## Abstract

Proposed a deep learning diagnostic framework that addresses the core limitation of single-modality ECG analysis. The pipeline transforms 1D ECG signals into four complementary 2D visual representations — Gramian Angular Fields (GAF), Markov Transition Fields (MTF), Recurrence Plots (RP), and Time-Frequency Representations (TFR) — which are fused and classified via a ResNet50 backbone.

---

## Key Contributions

- **Novel ECG-to-image transformation pipeline:** 1D signals → GAF, MTF, Recurrence Plot, and TFR representations
- **Multi-class diagnosis** of Myocardial Infarction (MI), Coronary Artery Disease (CAD), Congestive Heart Failure (CHF), and Arrhythmia
- **ResNet50 fusion backbone** achieving **99.75% classification accuracy** — among the highest reported for multi-class CVD detection in the literature
- **Explainable AI (XAI)** via Grad-CAM spatial saliency maps for per-class visual explanation
- **SHAP global attribution** for feature-level interpretability targeting physician trust and diagnostic transparency

---

## Core Technologies

### Multi-Modal Image Fusion
Combining diverse ECG image representations — GAF, MTF, Recurrence Plots, and TFR — for a comprehensive, holistic diagnostic input.

### Deep Learning Framework
State-of-the-art ResNet50 backbone for robust, multi-class classification of Cardiovascular Diseases with 99.75% accuracy.

### Explainable AI (XAI)
Grad-CAM visualizations and SHAP attribution provide clinical insights into the model's decision-making, building trust and transparency for real-world deployment.
